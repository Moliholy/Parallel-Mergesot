Parallel Programming for Multicore-Based Systems



Description
===========

This program is a parallelization of an improved version of Mergesort sorting algorithm using threads. It is written in C++, and uses the pthread library.


Compilation
===========

To compile just execute the following commands in a terminal:
	$ cd Debug
	$ make clean
	$ make all



Usage
=====

./ass6_parallel size numThreads
Where "size" is the total size of the array to test, and "numThreads" is the maximum number of threads that can be used during the execution.
